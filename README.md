# YMM4GameHub


### [最新バージョンをダウンロード](https://github.com/Dolphin-kun/YMM4GameHub/releases/latest)

## 概要
2026/4/1のエイプリルフールプラグインです。
YMM4でオンライン対戦ができるようになります。

## 利用規約
1. 本プラグインは商用・非商用問わず利用することができます。
2. クレジット表記は任意ですが、概要欄にプラグイン名や製作者の名前を書いていただけると泣いて喜びます。
3. 以下の行為は禁止とします。
	- 無断での二次配布
	- 公序良俗に反する行為
	- 製作者や第三者に不利益をもたらす行為
4. 本プラグインを利用したことによって発生したいかなるトラブル・損害についても、製作者は一切の責任を負いません。
5. 本プラグインはオンライン通信機能を含みます。通信環境やセキュリティには十分ご注意のうえ、自己責任でご利用ください。
6. 本プラグインを利用した配信・動画撮影は自由に行うことができ、収益化についても問題ありません。ただし、ゲームごとに利用規約が異なる場合がありますので、各ゲームの規約をご確認のうえご利用ください。
	7. 今後、利用状況の把握などの目的で、ニコニコモンズのコンテンツIDの記載をお願いする場合があります。



## 使い方
- YMM4GameHub プラグインをインストール
- ツールタブから「YMM4GameHub」を起動後、ゲームを選択
- 「マッチング開始」を押す
